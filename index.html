📖 はじめての方へ：このページの使い方（クリックで展開）

🎯 このページでできること

フォワーダー業務でよく発生する 見積もり作成 を、入力テンプレート・自動計算・プレビュー出力までワンストップで支援するツールです。 入力した内容は ブラウザ内 に保存され、再読み込みしても消えません（自動保存ON時）。

🚦 推奨の入力フロー

  1. ① 管理番号入力：仮 REF #・引き合い元・担当を入力します。
  2. ② 引き合い条件：インコタームズ → 輸出/輸入 → 輸送モード → ゾーンカードで作業範囲を確定します。
  3. ③ 貨物情報・サイズ計算：貨物名・重量・容積を入力。寸法明細から CBM / CW を自動計算できます。
  4. ④ 見積もりテーブル：支払い（仕入）と請求の数量・通貨・単価を入力し、乗せ幅・利益を確認します。
  5. ⑤ 特記事項・全体リマーク：自由記入欄と免責事項のプリセットで仕上げます。
  6. ⑥ プレビューボタンで Excel/CSV 形式に出力し、提出資料として活用します。

💾 保存・読み込み（画面上部のツールバー）

  • 🗂️ プリセット/一時保存：「輸入案件用」など名前付きで最大10件まで保存できます。
  • 自動保存：ONにすると入力から2秒後にブラウザへ自動退避します（同じブラウザのみ）。
  • 📤 出力 / 📥 読込：JSONファイルとして書き出し・読み込みが可能。他PCとの受け渡しに使います。

⌨️ 便利なショートカット

  • Ctrl+K：コマンドパレットを開く（セクションジャンプ等）
  • Tab：見積もりテーブルの最終列で押すと新しい行を追加（ON/OFF可）
  • ↑ / ↓：見積もりテーブルで同じ列の上下行へ移動
💡 各セクションには 「📖 詳しい使い方」 ブロックがあります。困ったときはまずそれを開いてください。

🗂️ 管理番号入力

案件を管理するための識別情報を入力します。仮 REF # は見積もり番号や問い合わせ番号など任意の形式で OK。後でフィルタリングや整理に使えます。

📖 詳しい使い方 — 管理番号入力

📌 各項目の意味

  • 仮 REF #：見積もり管理用の任意番号。社内ルール（例：Q-2024-001、YYYYMMDD-連番）に合わせて自由に入力します。後で プレビュー や CSV 出力 のヘッダにも表示されます。
  • 引き合い元名称：依頼元の会社名・代理店名。プリセット保存時の見出しとしても活用できます。
  • 担当：自社側担当者（営業窓口）の名前。複数案件を扱う際の検索キーになります。

✍️ 入力のコツ

  1. まず 仮 REF # を決めてから各項目を入力すると、後で案件単位の振り返りがラクになります。
  2. 未確定の場合は TBD や 未定 を入れておき、後で更新しても OK。
  3. 同じ顧客の見積もりが多い場合は、入力後すぐに 🗂️ プリセット/一時保存 で雛形として保存しておくと再利用できます。
💡 ここで入れた情報は プレビュー画面 の上部メタ情報や、JSON 出力ファイルのヘッダにも含まれます。

🚢 引き合い条件

まず① インコタームズを選び、次に② 輸出/輸入と③ 輸送モードを選択し、④ ゾーンカードでサービス範囲をピースごとに組み立ててください。

📖 詳しい使い方 — 引き合い条件

🚦 入力ステップ（上から順に）

  1. 🔖 インコタームズ：EXW / FOB / CIF / DDP などを選択。荷主と合意済みの条件を選びます。海上専用 (FAS/FOB/CFR/CIF) は Sea モード時のみ有効です。
  2. 方向：📤 輸出 または 📥 輸入 をクリック。ゾーンカードのデフォルトサブコン（日本協力会社 / 現地代理店）が自動で切り替わります。
  3. 輸送モード：🚢 Sea / ✈️ Air を選択。Sea を選ぶと FCL / LCL サブボタンが表示されます。LCL/AIR ではコンテナ欄が自動で非表示になります。
  4. オプション：🛡️ 保険付保あり をON にすると、見積もりテーブルに保険項目が自動追加されます（プリセット適用時）。

🧩 ゾーンビルダー（作業範囲の組み立て）

輸送ルートを 出発地 → 幹線輸送 → 到着地 の3ゾーンに分けて、自社/サブコンが担う作業ピースを チェックボックス で選びます。

  • ① 出発地側：🚚 集荷／🏬 倉庫・梱包・バンニング／🛃 通関（輸出）
  • ② 幹線輸送：キャリア名・POL（積出港）・POD（仕向港）を入力
  • ③ 到着地側：🛃 通関（輸入）／🏬 倉庫・デバン／📬 納入

各ゾーンカードの OFF/ON ボタン でカード自体の有効/無効を切り替えられます。ON 状態のピースだけが見積もりテーブルのプリセットに反映されます。

🏢 サブコン入力のコツ

  • ピースごとに サブコン名（協力業者） を入力できます。＋ ボタンで複数候補を追加可能。
  • デフォルトサブコン 欄に入力すると、そのゾーン内の全ピースに一括反映されます。
  • ここで入れたサブコン名は、見積もりテーブルの サブコン 列にも引き継がれます。

⚡ プリセット適用ボタン

🔧 この構成でプリセット適用 をクリックすると、現在のゾーン構成・輸送モード・保険ON/OFF に合わせた標準見積項目（運賃・THC・通関・配送など）が 見積もりテーブル に自動投入されます。 確認ダイアログで 「置換」「追加」「スキップ」 を選べます。

💡 インコタームズを変えると保険・通関の負担側が変わるため、変更時は ゾーンの有効化状態 も併せて見直してください。

📦 貨物情報・サイズ計算

貨物の基本情報を入力し、寸法から CBM・チャージャブルウェイトを自動計算できます。計算結果は下の見積もり表の数量欄へそのまま転記できます。

📖 詳しい使い方 — 貨物情報・サイズ計算

📦 基本情報グリッド（上段）

  • 貨物名：商品の通称や品目名（例：自動車部品、家具）。
  • コンテナ種類・サイズ／本数：FCL のときに必須。LCL や AIR を選んでいる場合は自動で非表示になります。
  • 重量（概算）／容積（概算）：手入力できますが、下の寸法計算結果から 📋 反映ボタン で自動上書きも可能。
  • 荷姿：カートン / パレット / ドラム 等。
  • 危険品・特殊貨物：危険品 / 冷蔵 / 冷凍 / 重量物 などを選択。該当時はクラス・UN番号などを 特記事項 に追記してください。
  • ⠿ ハンドルでドラッグすると入力欄の並び順を自由に変更できます（個人の好みに合わせてカスタマイズ）。

📐 寸法・重量の詳細入力（下段の計算表）

  1. ＋ 荷物の種類を追加 ボタンで行を増やし、貨物の種類ごとに 個数・長さ・幅・高さ・単重量 を入力します。
  2. 荷姿 欄でカートン/パレット等を選択。段積み 列のチェックで段積み可否を指定します。
  3. 📐 計算する をクリック → CBM 小計・総重量・チャージャブルウェイト が一括計算されます。

🔢 計算ロジック

  • CBM ＝ 長さ(cm) × 幅(cm) × 高さ(cm) × 個数 ÷ 1,000,000
  • 容積重量 (Sea LCL)：CBM × 1,000 kg（1 CBM = 1,000 kg 換算）
  • 容積重量 (Air)：(長さ × 幅 × 高さ) / 6,000 cm³ ÷ kg
  • チャージャブルウェイト：実重量と容積重量の 大きい方

🔁 計算結果の活用

  • 📋 貨物情報へ反映：計算後の重量・容積を上部の基本情報欄に上書きします。
  • 📋 見積もり行に数量を反映：既存の見積もり行に対し、計算で得た CBM や CW を 支払い数量 としてワンクリックで投入できます（行ごとに反映ボタンが表示されます）。
💡 詳細寸法を入れなくても、上の 概算 重量/容積 欄だけでも見積もりは作れます。提出時に精度を上げたいときに計算表を活用してください。
⚠️ 単位は cm / kg 固定です。インチ・ポンドで来た情報は事前に換算してから入力してください。

📋 見積もりテーブル

費用項目ごとに 支払い（仕入） と 請求 の数量・通貨・単価を入力します。③ 乗せ幅 を入力すると請求単価に自動加算、④ 利益 欄で粗利をリアルタイム確認できます。CSV インポートやカテゴリ順並び替えも利用可能です。

📖 詳しい使い方 — 見積もりテーブル

🧱 列の役割

  • カテゴリ：📦 海上運賃／🚚 国内輸送／🛃 通関費用 等。🔀 カテゴリー順 ボタンで自動グルーピング可能。
  • 課税：チェックを入れると項目名先頭に * が付き、課税対象であることを明示します。
  • 項目名：費用の名称（例：オーシャンフレート、THC、ドレージ）。
  • 支払い列（青）：仕入側の 数量・単位・通貨・単価。① CD は表示専用（編集不可）。
  • 請求列（緑）：② 請求側 の数量・通貨・単価。乗せ幅を入れると自動計算されます。
  • 利益計算列（黄）：③ 乗せ幅／④ 小計（請求数量 × 単価）／⑤ 利益（請求合計 − 支払い合計）／備考。
  • サブコン：協力業者名。引き合い条件のゾーンビルダーから自動引き継ぎされます。

🧮 自動計算ロジック

  • ② 請求単価 ＝ 支払い単価 ＋ ③ 乗せ幅（通貨が同じ場合）
  • ④ 小計 ＝ 請求 数量 × 請求 単価
  • ⑤ 利益 ＝ 請求合計 − 支払い合計（通貨をまたぐ場合は JPY 換算 行で確認）

⌨️ 行操作・ショートカット

  • ＋ 行を追加：末尾に新規行を追加。
  • 各行の ＋ ボタン：その行の 直下 に行を挿入。
  • ⠿ ハンドルを掴んでドラッグ → 行を上下に並び替え。
  • Tab キー：最終列で押すと新行を追加（チェックボックスでON/OFF可）。
  • ↑/↓ キー：同じ列内で上下の行へカーソル移動。
  • 選択行 → 📋 コピー / 🗑️ 削除：チェックボックスで複数行をまとめて操作。
  • 🧹 未入力行を削除：項目名も金額も空のグレーアウト行をまとめて削除。

💱 為替レート設定パネル

  • 支払いと請求で 通貨が異なる 場合、JPY 換算で合計と利益を表示できます。
  • 🔄 今すぐ取得：open.er-api.com から最新レートを取得。
  • 起動時に自動取得：チェックすると次回以降ページを開くと自動更新されます。
  • 手入力でレートを上書きすることも可能（社内レート運用時）。

📝 特記事項・補足（自由入力）

荷主からの特別な要望や制約事項を自由に記入します。指定船会社・B/L 条件・バン詰め場所・その他の注意点など、見積もりには含まれないが把握しておくべき情報を残す欄として活用してください。

📖 詳しい使い方 — 特記事項・補足

🎯 こんなときに使う

見積もり金額には反映しないが、案件遂行のうえで必須となる情報 をまとめておくフリーテキスト欄です。 営業引き継ぎ・オペレーション部門との情報共有・受注後のチェックリストとして活用してください。

📝 記入例

  • 荷主の工場での FC Stuff（バン詰め） 希望／時間指定あり
  • 特定船会社の指定あり（ONE / MSC / Maersk 等）
  • B/L 条件：Shipper's Order B/L / Telex Release 希望
  • HS コード未確定／関税分類は荷主側で対応予定
  • 到着時のフリータイム延長要望／DEM&DET の上限希望
  • 輸送中の 温度ログ や 振動計 設置の要否

📤 プレビューへの反映

この内容は プレビュー画面 の上部メタ情報や JSON ファイル にも含まれるため、社内引き継ぎ書としてそのまま使えます。 客先には見せたくない場合は、提出前に内容を整理してください。

📝 全体リマーク（条件・免責事項）

見積もり全体に共通して適用する条件・免責事項を入力します。プリセットボタン（有効期限・運賃変動条項・フリータイムなど）をクリックすると定型文を追加できます。提出前に必ず内容を確認してください。

📖 詳しい使い方 — 全体リマーク（条件・免責事項）

🎯 ここに入れるべき内容

客先に提示する 見積もりの前提条件・有効期限・免責事項 をまとめる公式テキスト欄です。 プレビュー / CSV 出力時の末尾にも表示され、書類としての体裁を整える役割を担います。

📌 主なプリセット項目

  • 有効期限：見積もりの提示期限（例：発行日より2週間）
  • 運賃変動条項：BAF / CAF / EBS など燃油・通貨サーチャージの変動を許容
  • フリータイム：DEM&DET の無料期間と超過時の費用負担
  • 燃油サーチャージ：適用月の変動を反映する旨
  • 為替レート条件：見積もり時のレート（TTS等）と変動時の取り扱い
  • 保険：付保の有無と責任範囲
  • 不可抗力：天災・港湾混雑時の免責

✍️ 使い方ステップ

  1. 該当する プリセットボタン をクリック → テキストエリアに定型文が挿入されます。
  2. 必要に応じて 日付・金額・期間 を編集（プレースホルダ部分を実値に置換）。
  3. 不要な文章は クリア ボタンまたは手動で削除。
  4. 文字数は右下の ○○文字 表示で確認できます。
💡 顧客や社風に合わせて文言を編集したら、その状態を 🗂️ プリセット/一時保存 で保存しておけば、次回案件で再利用できます。
⚠️ 免責事項は契約上の重要事項 です。営業所内の標準フォーマットと齟齬がないか、提出前に必ず確認してください。

📖 詳しい使い方 — スコーププリセット挿入

🔘 挿入モードの違い

  • 🔄 置換：現在の見積もり行を全て削除 し、新しいプリセットで入れ替えます。ゼロから作り直す場合に。
  • ➕ 追加：既存の行は残したまま、末尾にプリセット項目を追記します。追加サービスを足すときに。
  • スキップ：今回は挿入しない。設定（ゾーンや輸送モード）の変更だけ反映されます。

💡 おすすめの使い方

  1. 新規案件 → 「置換」 でテンプレートを敷いてから個別調整。
  2. 既存見積に保険・付帯サービスを足す → 「追加」。
  3. 条件だけ変えて項目は維持したい → 「スキップ」。
⚠️ 置換 は元に戻せません。必要なら事前に 🗂️ プリセット保存 で現状を退避してください。

📖 詳しい使い方 — CSV出力

📋 2種類のプリセット

  • 📄 客先提示用：請求側の項目のみ表示し、仕入価格・乗せ幅・利益・サブコン を非表示にします。そのまま顧客に提出可能。
  • 🏢 社内管理用：仕入・乗せ幅・利益・サブコンを含む 全列 を出力。営業/オペ間の引き継ぎや原価分析に。

🛠️ 個別カスタム

  1. 各列のチェックボックスを ON/OFF して、出力したい列だけを選択。
  2. すべて選択 / 解除 でまとめて切り替え可能。
  3. ⬇ ダウンロード を押すと CSV ファイルがブラウザでダウンロードされます。

🔁 再読み込み（CSV → このページ）

出力した CSV を 📂 CSV読み込み から再投入できます。チームメンバーが編集した CSV を取り込んで継続作業するなどに利用できます。

💡 課税対象の項目は項目名の先頭に * が付いた状態で出力されます。

📖 詳しい使い方 — プレビュー画面

👀 表示される内容

  • 上部：管理番号・引き合い元・担当・引き合い条件 のメタ情報
  • 中央：見積もり明細（テーブル形式・カテゴリ別グルーピング）

📤 出力ボタン

  • 📑 クリップボードにコピー：TSV 形式でコピー。Excel/Google Sheets に そのまま貼り付け 可能。
  • ⬇ CSV ダウンロード：列選択ダイアログを経由して CSV ファイルを保存。
  • 閉じる：このダイアログを閉じます（編集データは消えません）。

🎯 使い分けのコツ

  • メール本文に貼り付けたい → クリップボードコピー → Excel に貼り付け → 整形してから本文へ。
  • システムに取り込みたい → CSV ダウンロード（社内管理用プリセット）。
  • そのまま客先提出 → CSV ダウンロード（客先提示用プリセット）。
